# Netflix-Content-Analytics

This project is a hands-on Power BI dashboard built to explore and understand Netflix’s content library.
The idea behind this project was to take a real-world dataset and turn it into clear, meaningful insights using data visualization and storytelling.

By analyzing movies and TV shows across genres, ratings, countries, and timelines, this dashboard helps answer questions like:
What type of content does Netflix focus on?
Which countries contribute the most titles?
How has Netflix’s content grown over time?

What This Dashboard Shows
Total number of Movies and TV Shows
Movie vs TV Show distribution
Content ratings breakdown (TV-MA, TV-14, PG-13, etc.)
Country-wise content contribution
Top genres and categories
Content additions trend over the years
Key business takeaways in simple language

Tools & Skills Used
Microsoft Power BI
Power Query for data cleaning and transformation
DAX for calculated measures and KPIs
Data Visualization & Storytelling

Dataset Information
Source: Public Netflix dataset

Important columns used:
show_id
type (Movie / TV Show)
country
date_added
release_year
rating
listed_in (genre)
duration

Data Cleaning & Preparation
Before building the dashboard, the data was cleaned and prepared:
Removed duplicates and unnecessary rows
Handled missing and null values
Split columns like country and listed_in for better analysis
Standardized date formats
Created calculated columns and measures using DAX

Key KPIs
Total Titles
Total Movies
Total TV Shows
Total Genres
Country-wise content count
Year-wise content growth
