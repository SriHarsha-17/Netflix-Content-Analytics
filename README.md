# Netflix-Content-Analytics

## About the Project
This project is a hands-on **Power BI dashboard** built to explore and understand Netflix‚Äôs content library.

The idea behind this project was to take a real-world dataset and turn it into **clear, meaningful insights** using data visualization and storytelling.

By analyzing movies and TV shows across genres, ratings, countries, and timelines, this dashboard helps answer questions like:
- What type of content does Netflix focus on?
- Which countries contribute the most titles?
- How has Netflix‚Äôs content grown over time?

---

## What This Dashboard Shows
- Total number of Movies and TV Shows  
- Movie vs TV Show distribution  
- Content ratings breakdown (TV-MA, TV-14, PG-13, etc.)  
- Country-wise content contribution  
- Top genres and categories  
- Content additions trend over the years  
- Key business takeaways in simple language  

---

## Tools & Skills Used
- Microsoft Power BI  
- Power Query (data cleaning and transformation)  
- DAX (calculated measures and KPIs)  
- Data Visualization & Storytelling  

---

## Dataset Information
**Source:** Public Netflix dataset  

**Important columns used:**
- show_id  
- type (Movie / TV Show)  
- country  
- date_added  
- release_year  
- rating  
- listed_in (genre)  
- duration
